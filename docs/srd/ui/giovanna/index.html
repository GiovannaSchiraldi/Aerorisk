UI Mockups — Giovanna's Use Cases

Static, browser-viewable mockups for the three use cases in Giovanna_UseCases_Deliverable1. Each page lets you step through the use-case flow including alternative and exception paths.

UC-02

Log Out

Confirmation prompt, secure session termination, success toast, and the silent expired-session exception.

 UC-03

Update Profile

Pre-filled edit form, inline validation, password strength meter, error states, and success.

 UC-08

Export PDF Report

Results view, generation progress, browser download prompt, rendered PDF with NASA label, exceptions.
Tech note: these are intentionally framework-free HTML/CSS so they render in any browser without a build step — ideal for the Deliverable 1 presentation and the SRD appendix. They follow the AeroRisk design language and can be ported 1:1 into the React frontend.
